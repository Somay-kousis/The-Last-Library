# The Last Library

A small RAG project built while learning retrieval systems.

The idea is simple:

You enter a forgotten library and investigate what happened through recovered documents, journals, letters, and archive fragments.

Instead of searching a PDF, the retrieval system acts like an archive. Every question reveals different pieces of the story depending on what you ask.

## Why I Built This

While learning RAG, I realized most practice projects become:

* chat with PDF
* chat with website
* chat with documents

I wanted something that felt a little more alive.

The goal wasn't to build a game.

The goal was to understand:

* document loading
* chunking
* embeddings
* vector search
* retrieval
* prompting
* memory

while wrapping it inside a small mystery.

## Tech Stack

* Python
* LangChain
* FAISS
* HuggingFace Embeddings (MiniLM)
* Groq

## Current Features

* Document loading
* Chunking
* Local vector database
* Retrieval-based questioning
* Conversation summarization memory
* Archive-style responses

## Future Ideas

* Conditional retrieval
* Agentic workflows using LangGraph
* Hidden documents
* Multiple endings
* Theory validation

Built mainly as a learning project while studying RAG and Agentic AI.
